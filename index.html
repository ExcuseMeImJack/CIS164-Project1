CIS 164 - JavaScript Programming

What is this Course? | Nov 14, 2024

This course covers client-side Web scripting using JavaScript and other Web page development tools. It covers the programming techniques of JavaScript useful to Web designers. Students will learn to create interactive and dynamic Web pages.

Students Learning Outcomes (SLOs)
  1. Understand front end Web development
  2. Create JavaScript statements
  3. Work with JavaScript objects, functions, and events
  4. Learn how to debug and test a JavaScript application
  5. Learn how to script the DOM (Document Object Model) with JavaScript
  6. Work with images and form validation
  7. Understand what a JavaScript Library is
